Quickstart

Ready to get started? This page provides an introduction to Multigraph.

Hello, Multigraph!

A Multigraph is split into two parts: a display object (an html <div> element), and a "mugl" file (a specialized xml file). A minimal Multigraph web graph looks something like this:

graph.html

<head>
  <script type="text/javascript"
   src="//multigraph.github.io/js/multigraph-min.js"></script>
</head>

<div class="multigraph" data-src="graph.xml"
 data-width="500" data-height="400"></div>

graph.xml

multigraph

<mugl>
  <data> 
    <values>
      0, 3
      2, 4
      4, 3
      6, 4.5
      9, 0
      10,5
    </values>
  </data>
</mugl>

So what's going on here?

  1. First we integrate the Multigraph library by referencing its location in the head tag of our html file. Multigraph can be internally linked to your own copy of multigraph-min.js, or linked externally to //multigraph.github.io/js/multigraph-min.js.
  2. The next thing we do in our html file is instantiate the display object. To do this, we set the class attribute of a <div> element to multigraph. The data-src attribute links to our "mugl" file, graph.xml. The data-width and data-height attributes define the width and height of the graph on the page (these can also be defined externally in a CSS file).
  3. We then specify data values in our mugl file. In this case we use a <values> element to explicitly specify the values in the file. You can also specify a reference to a comma-separated values (csv) file, or even write your own web service that serves up data as it is needed. See the Mugl File Reference for information on how to spice up your graphs.
  4. Voil√†! We have a functioning Multigraph. Notice that your pointer indicates interactivity when you hover over the graph. The horizontal dimension supports zooming with the mouse wheel, and panning is achieved by holding left-click and moving the mouse. Note that you can only pan in one dimension at a time (either vertically or horizontally).

Head over to the Multigraph Tester to start experimenting on your own!
